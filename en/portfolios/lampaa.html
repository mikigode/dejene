Lampaa - Backend Developer

A startup providing electrical products

Lampaa

Lempa startup was an online store for buying electrical products that was accessible through the application platform for the convenience of users. As a backend developer, I was responsible for the API implementation and the admin part of this website.

One of the main challenges of this project was to create a user management system. For this purpose, a system was developed for user registration and login, which used user information encryption for greater security. In addition, there was a need to implement the ability to determine user access for different parts of the website, as well as the ability to determine the access level for administrators, which was well implemented.

Another challenge of this project was to create a system for managing products and shopping cart. For this purpose, an API was created to add, remove and update products and cart. Also, for the best experience of buyers, an online payment system was created using the payment gateway.

With this, this website became an easy and fast environment for its users to buy snow products. In addition, the implementation of the user management system, products and shopping cart, as well as the online payment system, helped to improve the user experience.

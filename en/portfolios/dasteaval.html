Dast E Aval

The DasteAval website is a dynamic news site that provides its audience with the latest news and important events on a daily basis. This website seeks to provide hot and practical news that can inform its audience about current world events

For the DasteAval website, there were the following technical challenges:
1- Increasing page loading speed: considering that this website is used to display news and visual content, increasing page loading speed has been one of the main challenges of this website. By optimizing codes, compressing images and using new technologies, we were able to improve page loading speed.
2- Data and database management: Considering that this website must receive new and updated news content continuously and store it in the database, data management has been a challenge. We managed this challenge by using powerful database and data management technologies.
3- Improving accessibility: due to the importance of the content presented on this website, to improve the accessibility of users with visual limitations, it was necessary to design a suitable user interface. By optimizing the user interface and using modern technologies, we were able to manage this challenge as well.

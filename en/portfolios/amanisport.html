Amanisport - Backend Developer

An Online Store

Amanisport

The Amani Sport website, which I developed as a back-end programmer, is an online sports equipment store. This website is developed using Laravel framework and MySQL database.

To implement this online store, I faced many technical issues. One of the main challenges was to create an inventory management system. For this purpose, a unique inventory algorithm was developed that automatically updated the inventory of each product, and out-of-stock products were removed from the site to avoid creating unpredictable orders.

However, among the other achievements of this project, we can mention designing an efficient and user-friendly interface and optimizing the performance of the website. Also, to increase the loading speed of the website pages, caching was used for the most visited pages. In addition, content optimization techniques and the use of SEO URLs were used to improve website SEO.

As a result, the programming of this sports equipment store website was successful by creating an optimal user interface, intelligent inventory management system and performance optimization.

Subrosa - Backend Develper

An online store

Subrosa

In Subrosa online store, using Laravel and Vue.js framework, a powerful and complete store system was built for selling gift accessories, rhinestones and luxury goods. In this project, we faced various technical challenges that we will explain below:

- Designing and implementing the product display and filtering system: Among the main challenges in the project was the implementation of the product display system based on different categories. To solve this challenge, we used the capabilities of Vue.js to implement advanced search and product filtering.

- Implementation of payment system and financial transactions: implementation of a safe and reliable payment system for customers.

- Designing and implementing a content management system: due to the large volume of products and the need for continuous and diverse updates, creating a content management system was necessary. In this project, we used the Laravel framework to implement a custom content management system.

Along with various technical challenges, this project was a series of important achievements for me

Some of the achievements of this site are:

1. Improving the user experience: by designing a beautiful and professional user interface, users could easily use this site to buy luxury and attractive goods online.

2. Increasing the speed of the website: by using optimization and reducing the size of the files, the loading speed of the website has been improved, which has increased the traffic of the website and also improved the user experience.

3. Creating security: By using various security methods, this site has provided a safe and flawless environment for buying luxury goods.

4. Technical support: By providing technical support to website users, problems and defects are quickly resolved and the shopping experience of users is improved.

5. Communication with customers: by creating contact forms and support via phone and email, users' communication with site managers is improved and their problems and needs are solved.

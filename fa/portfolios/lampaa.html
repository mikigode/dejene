لمپا

استارتاپ لمپا یک فروشگاه اینترنتی برای خرید محصولات برقی بود که برای راحتی کاربران، در بستر اپلیکیشن قابل دسترسی بود. به عنوان یک برنامه نویس بک اند، من مسئولیت پیاده‌سازی API و بخش ادمین این وبسایت را برعهده داشتم.

یکی از چالش‌های اصلی این پروژه، ایجاد یک سیستم مدیریت کاربران بود. برای این منظور، سیستمی برای ثبت‌نام و ورود کاربران توسعه داده شد که برای امنیت بیشتر از رمزنگاری اطلاعات کاربران استفاده شد. علاوه بر این، نیاز به پیاده‌سازی قابلیت تعیین دسترسی کاربران برای بخش‌های مختلف وبسایت و همچنین امکان تعیین سطح دسترسی برای ادمین‌ها بود که به خوبی پیاده‌سازی شد.

یکی دیگر از چالش‌های این پروژه، ایجاد سیستمی برای مدیریت محصولات و سبد خرید بود. برای این منظور، یک API برای افزودن، حذف و به‌روزرسانی محصولات و سبد خرید ایجاد شد. همچنین برای بهترین تجربه خریداران، یک سیستم پرداخت آنلاین ایجاد شد که از گیت‌وی پرداخت استفاده شد.

با این کار، این وبسایت برای کاربرانش به یک محیط آسان و سریع برای خرید محصولات برفی تبدیل شد. علاوه بر این، پیاده‌سازی سیستم مدیریت کاربران، محصولات و سبد خرید و همچنین سیستم پرداخت آنلاین، به بهبود تجربه کاربران کمک کرد.
